4. JENKINS INSTALLATION AND SETUP,EXPLORE THE ENVIRONMENT.
RUN JENKINS.EXE FILE
PRESS NEXT AND SELECT JENKINS DEFAULT PATH TO INSTALL AS SHOWN


PRESS NEXT AND SELECT RUN SERVICE AS LOCAL SYSTEM.

PRESS NEXT AND TEST PORT 8080 IS AVAILABLE SELECT NEXT.IF NOT AVAILABLE CHANGE PORT NO AND PRESS NEXT.


PRESS NEXT AND SELECT JAVA PATH IN THE SYSTEM 


PRESS NEXT AND INSTALL.
AFTER INSTALLATION LOG IN WITH INTIAL ADMIN PASSWORD FROM C:\ProgramData\Jenkins\.jenkins\secrets
SELECT RECOMMNENED PLUGINS TO INSTALL AFTER
CHANGE USERNAME AND PASSWORD TO ADMIN.
