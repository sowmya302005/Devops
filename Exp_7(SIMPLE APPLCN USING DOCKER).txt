6.	DEVELOP SIMPLE CONTAINERIZED APPLICATION USING DOCKER.
The docker file has to be created under the project
Dockerfile

FROM maven:3.9.6-eclipse-temurin-17 AS build
WORKDIR /app
COPY pom.xml .
RUN mvn dependency:go-offline -B
COPY src ./src
RUN mvn -B clean package -DskipTests
FROM tomcat:10.1-jdk17-temurin
RUN rm -rf /usr/local/tomcat/webapps/*
COPY --from=build /app/target/*.war/ usr/local/tomcat/webapps/ROOT.war
EXPOSE 8080
After creating file under projet folder ,Open terminal and set path of project.
